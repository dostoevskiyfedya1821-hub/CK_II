TITLE: SQLite FTS5 Extension
SOURCE: https://sqlite.org/fts5.html

FTS5 is SQLite's full-text search extension. It provides virtual tables and query capabilities optimized for searching text content rather than only matching exact values. In practical terms, FTS5 is the SQLite feature people reach for when they need tokenized text search, relevance-style matching behavior, or specialized indexing of documents.

The FTS5 documentation covers creation of FTS5 virtual tables, query syntax, tokenizers, prefixes, phrase search, boolean combinations, auxiliary functions, and configuration options. It is much more specialized than the ordinary SQL language pages because full-text search has its own indexing model and query semantics.

A key idea is that FTS5 tables are virtual tables. They are not ordinary row-store tables with a normal index attached. Instead, the module manages full-text indexing behavior tailored to document search. The page also explains that tokenization choices affect how text is split and matched, which has a major impact on retrieval quality.
